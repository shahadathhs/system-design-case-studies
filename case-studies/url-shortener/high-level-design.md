# High-Level Design

The system will consist of the following components:

1.  **Web Client:** A simple web interface for users to shorten URLs.
2.  **API Server:** A server to handle API requests for shortening URLs and redirecting users.
3.  **Database:** A database to store the mapping between short URLs and long URLs.
4.  **Counter Service:** A service to generate unique IDs for short URLs.
