# Requirements

## Functional Requirements

*   Users can post new content (e.g., text, images, videos).
*   Users can follow other users.
*   Users can see a feed of content from the users they follow.
*   The feed should be in reverse chronological order.

## Non-Functional Requirements

*   **High Availability:** The service should be highly available.
*   **Low Latency:** The feed should load quickly.
*   **Scalability:** The service should be able to handle a large number of users and posts.
*   **Durability:** Posts should not be lost.
